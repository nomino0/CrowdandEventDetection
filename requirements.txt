## Getting Started

1. Install the required dependencies:

2. Set up a Firebase project and configure the app to use the real-time database feature.

3. Set up a GCP account and create a new project.

4. Connect the Raspberry Pi to the GCP project and configure it to stream video to the cloud.

5. Train the AI model using the PyTorch library and the YOLOv5 algorithm.

6. Deploy the model to GCP and set up a scheduled task to run the model on a regular basis.

7. Build the mobile app using Flutter and connect it to the Firebase database to retrieve the data.
